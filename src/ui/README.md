# UI Module Documentation

This directory contains shared UI utilities, styles, and layout helpers for the gnostr-tui application.

## Key Features
- Defines color themes and style utilities for consistent look and feel.
- Provides layout helpers for arranging widgets and components.
- Contains reusable UI elements (e.g., status bars, popups).

## Main Components
- **SharedTheme**: Struct for managing color and style settings.
- **style**: Module with color, border, and text style helpers.
- **layout**: Utilities for flexible widget arrangement.

## Usage
- Import `SharedTheme` and style helpers to apply consistent theming.
- Use layout utilities to structure tab and component UIs.

See the source files for more details and examples of UI customization.
