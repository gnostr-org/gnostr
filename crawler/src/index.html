Gnostr Crawler Endpoints

Explore the available data endpoints for the Gnostr Crawler:

  • /relays.txt: A plain text list of detected relays.
  • /relays.yaml: Relays in YAML format.
  • /relays.json: Relays in JSON format.

These endpoints are served by the gnostr-crawler API server. To view them, ensure the crawler is running in serve mode (e.g., cargo run -- serve).
